# wcl-tooltip

## Installation

```
npm install https://github.com/kawausokun/wcl-tooltip
```

## Usage

```
window.addEventListener('load', function () {
  var tooltip = require('wcl-tooltip');
  tooltip.init('.tooltipped');
});
```

## Option

init options

```
{
    position: 'auto' // top, bottom, left, right
}
```
